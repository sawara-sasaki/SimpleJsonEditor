# Simple Json Editor
Simple json editor using web browser.


## Dependence
- golang environment


## Build
```bash
# Windows
make clean build-win
# Mac
make clean build-mac
# Linux
make clean build-linux
```

## Usage
Open the built application.

Then the page will open automatically in your web browser.
